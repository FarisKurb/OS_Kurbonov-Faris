# Задание

1. Создать вектор из **1 000 000 элементов** и заполнить его случайными числами от **1 до 100** с использованием `srand()` и `rand()`.
2. Найти сумму элементов:
   - **в одном потоке**,  
   - **в четырёх потоках**, разделив вектор на равные части.
3. Замерить время выполнения обоих вариантов с помощью `std::chrono::steady_clock`.
4. Корректно сравнить результаты, вывести:
   - сумму в одном потоке,
   - сумму в четырёх потоках,
   - время каждого варианта,
   - ускорение (**Speedup**).
5. Структура запуска потоков в обоих вариантах должна быть одинаковой для корректного сравнения времени.

---

# Реализация

- Используется `std::vector<int>` для хранения данных.
- Заполнение выполняется через `rand()`, инициализированную `srand()` от системного таймера.
- Функция `sumPart` суммирует диапазон `[begin, end)`.
- В однопоточном варианте:
  - используется массив потоков `std::thread workers_single[THREADS]`,
  - только поток `workers_single[0]` считает сумму,
  - остальные потоки пустые — для равной структуры запуска и join.
- В многопоточном варианте:
  - запускаются 4 рабочих потока,
  - каждому выдаётся свой диапазон элементов.
- Для измерения времени используется `steady_clock`, чтобы избежать скачков времени.
- Вывод форматируется через `std::fixed` и `std::setprecision(6)`.
- Проверяется совпадение сумм и рассчитывается ускорение.
